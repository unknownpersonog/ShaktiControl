## Installation

<p>To deploy ShaktiCtrl locally or on a server, follow these steps:</p>

<ol>
  <li><strong>Clone the Repository:</strong>
    <pre><code>git clone https://github.com/UnknownVPS/ShaktiCtrl.git
cd ShaktiCtrl</code></pre>
  </li>
  <li><strong>Install Dependencies:</strong>
    <p>Ensure you have Node.js and npm installed, then run:</p>
    <pre><code>npm install</code></pre>
  </li>
  <li><strong>Configure Environment Variables:</strong>
    <p>Set up your <code>.env</code> file with necessary configurations (e.g., API keys, database connections).</p>
  </li>
  <li><strong>Run the Application:</strong>
    <pre><code>npm start</code></pre>
  </li>
</ol>

## How It Works

<p>ShaktiCtrl is designed to simplify the management of UnknownVPS services by offering a straightforward, accessible interface. It connects seamlessly with <strong>YogAPI</strong> to handle backend operations and data flow. Users can:</p>
<ul>
  <li>Manage server instances.</li>
  <li>Monitor real-time resource consumption.</li>
  <li>Access logs and detailed metrics.</li>
  <li>Adjust settings and configurations through an intuitive UI.</li>
</ul>

<p>The admin panel adds an extra layer of control for administrators to manage client accounts, oversee system status, and maintain optimal performance.</p>
